# Node.js Food App APIs

This project is a Node.js backend application that provides REST APIs for a food application.

## Tech Stack
- Node.js
- Express.js
- MongoDB / MySQL
- JWT Authentication

## Features
- User authentication
- Vendor management
- Food / product APIs
- Role-based access
- RESTful API structure

## API Type
Backend only (API collection)

## How to Run
1. Clone the repository
2. Run `npm install`
3. Create `.env` file
4. Run `npm start`

## Purpose
This project is created to practice backend development and API design.
